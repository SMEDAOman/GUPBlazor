# Welcome to the Oman Gov.om component library!

These Storybook docs contain detailed information on how to install and use the Gov.om component library.

These are web components that work with any framework, with styling built into each component.

Please choose an item you need in the sidebar to the left (press **S** if you don't see it).

## What's Included

- **Web Components**: Modern, framework-agnostic components that work with any JavaScript framework
- **React Components**: Pre-built React wrappers for seamless integration
- **Angular Components**: Angular-specific implementations with full TypeScript support
- **Design Tokens**: Consistent design variables for colors, typography, and spacing
- **Icon Library**: Comprehensive set of government-appropriate icons
- **Accessibility**: WCAG-compliant components with built-in accessibility features
- **Documentation**: Comprehensive guides and API documentation

## Getting Started

The Gov.om component library is designed to provide a consistent user experience across all Oman government digital services.

📖 **For installation instructions, see the [Installation](?path=/docs/installation--documentation) page**

📋 **For examples and tips, see the [Usage](?path=/docs/usage--documentation) page**

📋 **For Figma components and templates, download the Figma file from the Figma community page**

### Browser Support

Gov.om components support all modern desktop and mobile browsers including Chrome, Firefox, Safari, and Edge.

## Framework Integration

Choose your preferred framework for detailed integration guides:

- 🌐 **Vanilla JavaScript**: Use web components directly - see the No Bundler page
- ⚛️ **React**: Dedicated React package with TypeScript definitions - see the React Integration page
- 🅰️ **Angular**: Full Angular integration with modules and services - see the Angular Integration page
- 🖼️ **Other Frameworks**: Web components are framework-agnostic - see the Frameworks page
