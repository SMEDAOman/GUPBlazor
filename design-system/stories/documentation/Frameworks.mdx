# Integrating with your framework

See these docs for integration of the UI library with your framework:

1. [React](https://react.dev/reference/react-dom/components#custom-html-elements). Note that starting from v19 React provides significantly better compatibility with Web components than React &lt;19. Check also [a dedicated React page](?path=/docs/using-with-react--documentation) for React integration.
2. [Angular](https://angular.dev/guide/elements). Check also a [dedicated Angular page](?path=/docs/using-with-angular--documentation) for integration with Angular forms.
3. [Vue](https://vuejs.org/guide/extras/web-components)

Our team has prepared a few playgrounds with examples of the UI library components used with various frameworks. Take a look:

1. [React &lt;19](https://codesandbox.io/p/sandbox/gup-components-react-l62tqd)
2. [Angular](https://github.com/ekateriinal/gup-components-angular)
3. [AngularJS](https://codesandbox.io/p/github/ekateriinal/gup-components-in-angulajs)
4. [Raw HTML](https://codepen.io/certainlyakey/pen/LYKxrjw?editors=1000)
