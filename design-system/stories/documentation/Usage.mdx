# Usage

## Utility components

Utility components are components that don't have much styling in them but that still solve small everyday tasks.

### Managing spacing

Utilise `gup-track` utility component to manage spacing consistently with the GUP design system. It allows you to set space with the `gap` property without brand violations.

### Providing screenreader-only access

Use `gup-screenreader-text` utility component to hide text on all devices except screen readers.

### Icons

The GUP design system uses [Material Icons](https://marella.me/material-design-icons/demo/svg/). You can find all the available icons on the `@material-design-icons/svg` package page or in the [Components/Icon/All icons story](?path=/story/components-icon--all-icons).

In your project you can utilise the icons with the `gup-icon` component or by importing them directly from the `@material-design-icons/svg` package.
