# Installation – Web components

1. Run `npm install @govom/components` in your project.
2. Add a script to your HTML file (modify the URL to point at where the `dist` folder of the package is hosted):

```html
<!-- Add only the components you need, include their dependencies too -->
<script type="module" src="https://myurl.com/gup-components/dist/track.js"></script>

<!-- OR add all components at once (not recommended for performance reasons) -->
<script type="module" src="https://myurl.com/gup-components/dist/components.js"></script>

<!-- OR use the experimental autoloader that lazy-loads only the components appearing on the page -->
<script>
  const GUP_AUTOLOADER_SOURCE_DIR = "https://myurl.com/gup-components/dist/";
</script>
<script src="https://myurl.com/gup-components/dist/autoloader.js" defer></script>
```

See [the dedicated page](?path=/docs/using-without-bundler--documentation) for use cases when bundler is not used.

3. Add styles to your HTML file:

```html
<link rel="stylesheet" href="/node_modules/@govom/components/dist/styles.css">
```

You can also import the styles into a PostCSS or Sass-enabled file:

```css
@import "@govom/components/styles";
```

4. Start using the components in your HTML.

# Installation - React

See [the dedicated React page](?path=/docs/using-with-react--documentation) for more details.

1. Run `npm install @govom/react` in your project. Don't forget to import the styles too.
2. Import the React components using `import { Button } from '@govom/react';` syntax.

# Installation - Angular Forms compatibility layer

See [the dedicated Angular page](?path=/docs/using-with-angular--documentation) for more details.

1. Run `npm install @govom/angular` in your project. Don't forget to import the styles too.
2. Import a compatibility module with `import { GupFormsCompatModule } from '@govom/angular';` to a component or module.
